<h2>Install</h2>

```sh
git clone https://github.com/chrisjameschamp/reverse-PDF.git
```

```cd reverse-PDF```

```./reversepdf.py "/path/to/your/pdf/file/"```

The resulting PDF will be placed on your Desktop... That's it!
